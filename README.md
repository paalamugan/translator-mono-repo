# Translator Application
It helps to translate from one language to multiple languages.
    
## Get Started

- Node Engines
 - node: v17.1.0
 - npm: 8.1.2

- Install Dependecies

    ```sh
    npm run install
    ```

- To Development

    ```sh
    npm run dev
    ```

- To Build
    ```sh
    npm run build
    ```

- To Production
    ```sh
    npm run start
    ```

**Deploy this application in vercel platform**

- Deploy in locally
    ```sh
    npm run local
    ```

- Deploy in production
    ```sh
    npm run deploy
    ```
